# 各种游戏排行榜

## 游戏榜单

> https://le3-api.game.bilibili.com/pc/game/ranking/page_ranking_list

_请求方式：GET_

**请求参数：**

| 参数名       | 类型 | 内容     | 必要性 | 备注                                                                         |
| ------------ | ---- | -------- | ------ | ---------------------------------------------------------------------------- |
| ranking_type | num  | 排序类型 | 必要   | 1:热度榜</br>2:口碑榜</br>5:预约榜</br>6:新游榜</br>7:`B 指榜`</br>11:端游榜 |
| page_num     | num  | 页数     | 非必要 |                                                                              |
| page_size    | num  | 页大小   | 非必要 |                                                                              |

**json 回复：**

<div id="samePart"></div>
根对象:

| 字段       | 类型 | 内容       | 备注                         |
| ---------- | ---- | ---------- | ---------------------------- |
| code       | num  | 返回值     | 0：成功<br/>-101：账号未登录 |
| message    | str  | 验证信息   | 成功或用户尚未登录           |
| request_id | str  | 返回 id    | 随机生成                     |
| ts         | num  | 当前时间戳 |                              |
| data       | str  | 信息本体   |                              |

`data`对象：

> 根据`ranking_type`的值确定

| 字段              | 类型   | 内容       | 备注                |
| ----------------- | ------ | ---------- | ------------------- |
| hot_list          | arrary | 返回值     | `ranking_type` 为 1 |
| good_commend_list | arrary | 验证信息   | `ranking_type` 为 2 |
| new_game_list     | arrary | 返回 id    | `ranking_type` 为 5 |
| bindex_list       | arrary | 当前时间戳 | `ranking_type` 为 6 |
| client_game_list  | arrary | 信息本体   | `ranking_type` 为 7 |
