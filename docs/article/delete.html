# 删除专栏

https://member.bilibili.com/x/web/article/deleteopen in new window

请求方式：POST

认证方式：Cookie（SESSDATA）

正文参数（application/x-www-form-urlencoded）：

参数名类型内容必要性备注
aidnum文章cvid必要
csrfstrCSRF Token（位于cookie）必要

json回复：

根对象：

字段类型内容备注
codenum返回值0：成功
-101：账号未登录
-111：csrf校验失败
-400：请求错误
-404：无此项
37104：无权限操作他人专栏
messagestr错误信息默认为0
ttlnum1

示例：

curl 'https://member.bilibili.com/x/web/article/delete' \
  --data-urlencode 'aid=114514' \
  --data-urlencode 'csrf=xxx' \
  -b 'SESSDATA=xxx'
查看响应示例：
{
  "code": 0,
  "message": "0",
  "ttl": 1
}
