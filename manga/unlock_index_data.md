## 创建异或表

在本话网址（ https://manga.bilibili.com/mc26731/329893 ）中包含了两个重要参数

- mc=26731

- ep=329893

```javascript
function f(mc, ep) {
    var n = new Uint8Array(new ArrayBuffer(8));
    n[0] = mc,
    n[1] = mc >> 8,
    n[2] = mc >> 16,
    n[3] = mc >> 24,
    n[4] = ep,
    n[5] = ep >> 8,
    n[6] = ep >> 16,
    n[7] = ep >> 24;
    return n;
}
```

mc、ep向右移位并储存到n的0-7位，得到的n即为异或所需的 array 。

