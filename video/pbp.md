## 获取弹幕趋势顶点列表

> http://bvc.bilivideo.com/pbp/data

*请求方式：GET*

**url参数：**

| 参数名 | 类型 | 内容     | 必要性 | 备注 |
| ------ | ---- | -------- | ------ | ---- |
| cid    | num  | 视频CID  | 必要   |      |
| aid    | num  | 稿件avID | 非必要 |      |
| bvid   | num  | 稿件bvID | 非必要 |      |

**json回复：**

根对象：

| 字段     | 类型 | 内容         | 备注                         |
| -------- | ---- | ------------ | ---------------------------- |
| step_sec | num  | 采样间隔时间 | 单位为秒<br />由视频时长决定 |
| tagstr   | str  | ？？？       | 作用尚不明确                 |
| events   | obj  | 数据本体     |                              |
| debug    | str  | 调试信息     | json字串                     |

`events`对象：

| 字段    | 类型   | 内容       | 备注 |
| ------- | ------ | ---------- | ---- |
| default | array | 顶点值列表 |      |

`events`对象中的数组：

| 项   | 类型 | 内容      | 备注                             |
| ---- | ---- | --------- | -------------------------------- |
| 0    | num  | 顶点1     |                                  |
| n    | num  | 顶点(n+1) | 顶点个数由视频时长和采样时间决定 |
| ……   | num  | ……        | ……                               |

**示例：**

获取视频`CID=3724723`的获取弹幕趋势顶点列表

```shell
curl -G 'http://api.bilibili.com/pbp/data' \
--data-urlencode 'cid=3724723'
```
