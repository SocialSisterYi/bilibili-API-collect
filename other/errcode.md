# 公共错误码

下表为大部分接口返回值中`code`字段的值中公共的错误代码

## 权限类：

| 代码 | 含义                     |
| ---- | ------------------------ |
| -1   | 应用程序不存在或已被封禁 |
| -2   | Access Key错误           |
| -3   | API校验密匙错误          |
| -4   | 调用方对该Method没有权限 |
| -101 | 账号未登录               |
| -102 | 账号被封停               |
| -103 | 积分不足                 |
| -104 | 硬币不足                 |
| -105 | 验证码错误               |
| -106 | 账号非正式会员或在适应期 |
| -107 | 应用不存在或者被封禁     |
| -108 | 未绑定手机               |
| -110 | 未绑定手机               |
| -111 | csrf 校验失败            |
| -112 | 系统升级中               |
| -113 | 账号尚未实名认证         |
| -114 | 请先绑定手机             |
| -115 | 请先完成实名认证         |
